# Logo Files

Đặt 2 file logo vào thư mục này:

1. **icool-logo.png** (hoặc .jpg, .svg) - Logo ICOOL
2. **spaa-logo.png** (hoặc .jpg, .svg) - Logo SPAA

## Định dạng hỗ trợ:
- PNG (.png)
- JPG/JPEG (.jpg, .jpeg)
- SVG (.svg)

## Kích thước khuyến nghị:
- Chiều cao: ~48px (h-12)
- Chiều rộng: Tự động (max 150px)
- Tỷ lệ: Giữ nguyên tỷ lệ gốc

## Lưu ý:
- Tên file phải chính xác: `icool-logo.png` và `spaa-logo.png`
- Nếu dùng định dạng khác (.jpg, .svg), cần cập nhật đường dẫn trong `index.html`
